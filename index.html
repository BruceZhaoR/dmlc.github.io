Welcome to DMLC

DMLC is a community hosting and developing portable, scalable and reliable libraries for distributed machine learning. Contributors come from (in alphabetical order) Baidu, CMU, HKUST, Microsoft, NYU, UW, ... We are looking forward new contributors and projects to join it.

Machine Learning Libraries

  • Boosted Trees (GBDT or GBM): XGBoost
  • Linear method (Sparse logistic regression, and others..): SGD, L-BFGS, Block CD
  • Deep learning CXXNET Minerva
  • Topic Model LDA

System Components

  • I/O (local disk, HDFS, Amazon S3) and Job launchers (Yarn, MPI, ...) dmlc-core
  • Data communication
    • rabbit for BSP synchronization(allreduce)
    • parameter server for asynchronous parameter update
